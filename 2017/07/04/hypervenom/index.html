浑身是戏 － 新毒蜂评测

Jul 4, 2017
chen

‘低帮顶级和高帮超顶配置完全相同’，‘Poron鞋面爆射神器’，‘超高响应大底灵活轻便’。新毒蜂刚刚推出时，各大论坛里可以讲是好评如潮。虽然走了内马儿这个当家代言，莱万、奥巴梅扬、姆巴佩等一票实力+天赋爆表的前场球员还是为这款新作赚足眼球。然而，毒锋三面世的前两个月，小编一直是拒绝的 – 首发配色这么丑，没法儿抱回家睡啊… 小编有个习惯，新鞋到手一般要先抱着睡两天培养下感情，感情沟通好了才能带上训练场，然后跑个十来圈儿踩开了就算开始服役了。首发红配绿，真的是下不去手啊。

转折出现在两个月之前，小编的ex主力战靴高帮毒锋2.5ag翔一般的配色那一双，由于多处开胶和鞋面破损终于退役了。之后尝试了Adidas Purechaos，X16.1和一双老鞋里格拉，不是太软就是太硬，不是太松就是太紧，始终找不到一双中意合适的主力战靴。终于，看到姆巴佩上脚这双蓝白配色简单清澈的新毒蜂脚下生花时，小编没再犹豫一把拿下。

新毒蜂从鞋身都鞋底都是噱头满满：整体飞线+热熔涂层和acc的鞋面材质里面还镶嵌Poron泡沫，延伸在外的飞线作为鞋带孔号称能更好调节鞋型，HyperReactive FG大底六角钉和刀钉的强悍组合，两侧不均衡的鞋领设计提供更好锁定。对于球鞋的科技元素，小编一贯还是持保守态度，试过再说的，下面我们就来看看这些噱头实战性能如何。

飞线鞋面：Nike号称为新毒蜂装上了3米的飞线，足足是前作的三倍。小编这双鞋其实选得尺码比较宽松（下文会专门讲尺码），所以刚上脚锁定感并不是很强烈。通过收紧裸露在外的飞线鞋带控，鞋身确实有一定程度的收紧，脚弓脚背的锁定效果都比较理想。关于飞线另一点不得不提的就是断裂的风险，这也是小编第一次看毒锋谍照时最大的忧虑。事实证明，so far so good，小编踢球风格比较凶悍，拼抢多、对脚多、还犯规多，这双毒锋服役两个多月，大概踢了十来场球，飞线看起来还是十分坚固，没有明显磨损迹象的。

鞋身Poron泡沫：号称打门触球时会瞬间变硬的Poron泡泡，概念听起来很像最近有点火的某球王代言的护腿板（跑个题这个板儿实战效果有点菜的）。不过小编通通不信的，小编只是感觉鞋面和小凸起都很软，触球反馈很直接，发力干脆不拖泥带水，Poron小凸起客观上也起到了不错的摩擦效果，对发力长传、打门的精准性很有辅助。内嵌泡沫的织物鞋面并不算特别轻薄，所以保护性还是过关。

Hyper-Reactive超高响应FG大底：讲真，拿一双FG来踢人草，小编心里是敲小鼓的。踢遍上中下人草之后，小编觉得：真是比ag-pro强太多了！Pedax结合尼龙的材料提供了足够的强度，完全不像ag-pro那样软塌塌的会卸力。超高响应大底后钉虽然看着有点长，好在连接鞋钉的加强梁和大底中部的流线结构都在客观上减短了鞋钉高度。如果不是体重特别大或者球风左右右扭闪断腰的朋友，拿这FG应付中等人草还是轻轻松松的。

新Dynamic Fit不均衡动态鞋领：据说这种新的高帮样式是根据脚踝内外侧本身的不平衡属性设计的，不过在小编十来场的实战、跑动中，感觉跟毒锋2.5代的高帮并无明显区别。经常有队友问小编，高帮球鞋可以保护脚踝吗，可以提供支撑吗？小编的亲身感受是，除了隔离人草颗粒进入鞋仓，高帮还真没什么明显作用。

毒锋三的鞋型，尺码与前作基本一致，横向对比其它系列的话，鞋长与传奇、鬼牌基本一致，前掌空间大于传奇，小于鬼牌。小编传us10/jp 280的传奇六完美fit，这次毒锋选择了相同的尺码，感觉包裹感弱于k皮传奇，一方面前掌空间要大些，另一方面织物面料毕竟不像真皮般贴合脚型。之前在实体店试鞋时感觉9.5太挤，10码又不够包裹感，犹豫之后还是选了10码，然后换上传奇四的poron鞋垫，再通过鞋身的飞线系统收缩下鞋面，最终实战效果确实还是不错，脚面没有压迫感，也没有过多鞋仓内的滑动。重量方面，这对10码毒蜂，单只重量在200g上下，上脚感觉得出来比前作轻（也有可能因为ex是双ag）。

Bonus tip，这个配色是意产！（前几个配色都是波产）
